# vaidehi-rp.github.io

A clean, responsive portfolio website built for GitHub Pages.

## Quick Start

1. Clone the repository:
   ```bash
   git clone https://github.com/vaidehi-rp/vaidehi-rp.github.io.git
   cd vaidehi-rp.github.io
   ```
2. Open `index.html` in your browser (or use VS Code Live Server).
3. Edit portfolio content in:
   - `index.html` (text/content/links)
   - `styles.css` (design/colors/layout)
   - `script.js` (theme + mobile nav behavior)

## Publish to GitHub Pages

For a `username.github.io` repository, GitHub Pages serves from the root automatically:

1. Commit your changes:
   ```bash
   git add .
   git commit -m "Update portfolio content"
   ```
2. Push to `main`:
   ```bash
   git push origin main
   ```
3. Open repository settings:
   - `Settings -> Pages`
   - Source: **Deploy from a branch**
   - Branch: `main` / `/ (root)`
4. Save and wait for deployment.
5. Your site will be available at:
   - `https://vaidehi-rp.github.io/`
